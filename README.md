# hanzi-chai/svg

`hanzi-chai/svg` 是一个用于高效校对「文」数据库中笔画数据信息的工具。

# 安装

首先确保安装好 Node.js 并克隆本仓库。

1. 将 `glyphs.ttf` 复制到 `/assets/` 下；
2. `npm install` 安装依赖；
3. `npm run glyphs` 生成字体文件缓存；
4. `npm run build` 生成前端页面；
5. `npm start`；
6. 打开 http://localhost:4000/。

# 运行

1. 在左侧列表中选取要修改的字；
2. 调整右方表单；
3. 将该字标记为已完成；
4. 点击保存，上传到服务器。
